# First Customer Checklist

Use this checklist to onboard the first real customer.

- [ ] Create organization and admin account
- [ ] Add apartments/properties
- [ ] Add initial reservations
- [ ] Assign managers to properties
- [ ] Set preferred language (`ro` / `ru` / `en`)
- [ ] Validate calendar + cleanings workflow
- [ ] Export CSV (reservations/properties/clients/cleanings)
- [ ] Test support + feedback flow
- [ ] Confirm backup and launch preflight (`docs/LAUNCH.md`)
